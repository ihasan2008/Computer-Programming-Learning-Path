# > (1) VS Code

VS Code (Visual Studio Code) হলো Microsoft-এর তৈরি একটি জনপ্রিয়, হালকা (Lightweight), দ্রুত (Fast) এবং শক্তিশালী (Powerful) Source Code Editor, যা বিভিন্ন Programming Language-এ Code লেখা, সম্পাদনা (Edit), Debug, Run এবং Project পরিচালনার জন্য ব্যবহৃত হয়।

সহজ ভাষায়:

VS Code হলো এমন একটি Code Editor, যেখানে Programmer-রা HTML, CSS, JavaScript, Python, Java, C, C++, PHP এবং আরও অনেক Programming Language-এ Code লিখতে, চালাতে এবং Debug করতে পারে।

## What is VS Code?

VS Code একটি Cross-platform Source Code Editor, যা Windows, macOS এবং Linux-এ চালানো যায়।

```text id="vsc01"
Programmer
      |
   VS Code
      |
Write & Edit Code
      |
Build Software
```

VS Code বর্তমানে বিশ্বের অন্যতম জনপ্রিয় Code Editor।

## Why is VS Code Important?

VS Code গুরুত্বপূর্ণ কারণ:

* দ্রুত কাজ করে।
* ব্যবহার করা সহজ।
* অনেক Programming Language সমর্থন করে।
* হাজার হাজার Extension ব্যবহার করা যায়।
* Debugging সুবিধা রয়েছে।
* Git Integration রয়েছে।
* Professional Software Development-এর জন্য উপযোগী।
